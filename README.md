爱播支持列表播放、全屏播放、重力自动旋转、清晰度切换、播放引擎切换、自定义主题颜色等。
## 特点
* 全屏重力感应自动旋转
* 清晰度切换(高清/超清/蓝光)
* 自定义播放引擎(ijkPlayer/ExoPlayer/MediaPlayer)
* 视频列表播放(ListView/RecyclerView)
* 自定义主题颜色
